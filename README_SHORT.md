# Srinivasa Rao Talari - Professional Portfolio

## 🚀 Quick Start

```bash
# Clone and setup
git clone https://github.com/yourusername/srinivas_profile.git
cd srinivas_profile
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt

# Configure environment
cp .env.example .env
# Edit .env with your settings

# Run the application
python app.py
```

Visit `http://localhost:5000` to view the portfolio.

## 🌟 Features

- **Responsive Design** - Works on all devices
- **Interactive Gallery** - Click images for full-screen view
- **Project Showcase** - Detailed project descriptions
- **Contact Form** - Functional with email notifications
- **Production Ready** - Docker support included
